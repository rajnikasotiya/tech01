### 1. Question Refinement Pattern

**Prompt:**
“You are an AI assistant using the Question Refinement Pattern. When I ask a vague or broad question, first refine it into a clearer, more specific question. Then provide the improved question and answer it. Repeat this process for multiple questions.”

**Sample Output:**

**User Question 1:** “Tell me about AI.”

**Refined Question:**
“What are the main applications of Artificial Intelligence in healthcare, finance, and education?”

**Answer:**
Artificial Intelligence is widely used in healthcare for medical diagnosis, drug discovery, and patient monitoring. In finance, AI helps with fraud detection, algorithmic trading, and customer support chatbots. In education, AI powers personalized learning platforms, automated grading, and virtual tutoring systems.

---

**User Question 2:** “How do I learn programming?”

**Refined Question:**
“What is a beginner-friendly roadmap for learning Python programming in 3 months?”

**Answer:**
Start with Python basics such as variables, loops, functions, and data structures during the first month. In the second month, practice problem-solving and build small projects like calculators or to-do apps. In the third month, learn libraries such as Pandas and Flask, and create a portfolio project to strengthen your skills.

---

**User Question 3:** “What is cloud computing?”

**Refined Question:**
“How does cloud computing work, and what are the differences between IaaS, PaaS, and SaaS?”

**Answer:**
Cloud computing provides computing resources such as storage, servers, and software over the internet. IaaS offers virtual infrastructure, PaaS provides development platforms, and SaaS delivers ready-to-use software applications through a web browser.

### 2. Cognitive Verifier Pattern

**Prompt:**
“You are an AI assistant using the Cognitive Verifier Pattern. For every complex question, first break the task into smaller verification steps. Check assumptions, verify reasoning, and then provide the final answer with explanations.”

**Sample Output:**

**User Question:**
“Should a company migrate its applications from on-premise servers to the cloud?”

**Verification Steps:**

1. Identify the company’s current infrastructure costs.
2. Evaluate scalability and performance needs.
3. Compare security and compliance requirements.
4. Estimate migration complexity and downtime risks.
5. Analyze long-term maintenance and operational benefits.

**Final Answer:**
Migrating to the cloud can reduce infrastructure maintenance costs and improve scalability. Cloud platforms also provide flexible resource allocation and disaster recovery capabilities. However, migration may introduce short-term costs, training requirements, and security considerations. A hybrid approach is often recommended for organizations with strict compliance needs or legacy systems.

---

**User Question:**
“What is the best machine learning model for fraud detection?”

**Verification Steps:**

1. Determine whether the dataset is balanced or imbalanced.
2. Identify the type of fraud patterns in the data.
3. Compare supervised and unsupervised learning methods.
4. Evaluate performance metrics such as precision and recall.
5. Test models using cross-validation.

**Final Answer:**
For fraud detection, ensemble models such as Random Forest and XGBoost are commonly effective because they handle complex patterns and imbalanced datasets well. Deep learning methods may work better for large-scale transaction data. Precision and recall are more important evaluation metrics than accuracy because fraud datasets are usually highly imbalanced.
