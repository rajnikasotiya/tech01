### 2. Sample Prompts Used

1. “Read all uploaded images and create resized copies in multiple dimensions suitable for social media, thumbnails, presentations, and web use.”

2. “Generate three resized versions of each image: small (256x256), medium (512x512), and large (1024x1024). Preserve image quality while maintaining aspect ratio.”

3. “Save each transformed image with descriptive filenames that include the original filename and output size.”

4. “Create a CSV catalog listing the original image name, transformed image filename, dimensions, file size, and transformation applied.”

5. “Package all generated images and the CSV catalog into a ZIP file for download.”
