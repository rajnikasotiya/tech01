## 1. Few-Shot Prompt for a Complex Task

### Task: Classifying Customer Support Tickets

### Prompt

“You are an AI support assistant. Classify customer support messages into categories: Billing Issue, Technical Problem, Account Access, or General Inquiry.

Example 1:
Input: ‘I was charged twice for my subscription this month.’
Output: Billing Issue

Example 2:
Input: ‘The application crashes whenever I upload a document.’
Output: Technical Problem

Example 3:
Input: ‘I forgot my password and cannot log into my account.’
Output: Account Access

Now classify this message:
‘My payment went through, but my premium features are still locked.’”

### Sample Output

Billing Issue

## 2. Few-Shot Prompt with Intermediate Sub-Steps

### Task: Extracting Action Items from Meeting Notes

### Prompt

“You are an AI meeting assistant. Perform the following steps:

1. Identify important tasks mentioned in the text.
2. Identify the owner of each task.
3. Identify deadlines if available.
4. Present the final answer in a table.

Example 1:

Input:
‘Raj will prepare the API documentation by Friday. Aman will test the login module next week.’

Step 1: Tasks Identified

* Prepare API documentation
* Test login module

Step 2: Owners Identified

* Raj
* Aman

Step 3: Deadlines Identified

* Friday
* Next week

Final Output:

| Task                      | Owner | Deadline  |
| ------------------------- | ----- | --------- |
| Prepare API documentation | Raj   | Friday    |
| Test login module         | Aman  | Next week |

Now process this input:

‘Neha will deploy the fraud detection model by Monday. Ravi will validate the staging environment before production release.’”

### Sample Output

Step 1: Tasks Identified

* Deploy fraud detection model
* Validate staging environment

Step 2: Owners Identified

* Neha
* Ravi

Step 3: Deadlines Identified

* Monday
* Before production release

Final Output:

| Task                         | Owner | Deadline                  |
| ---------------------------- | ----- | ------------------------- |
| Deploy fraud detection model | Neha  | Monday                    |
| Validate staging environment | Ravi  | Before production release |
